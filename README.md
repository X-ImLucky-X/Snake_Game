# 🐍 Snake Game

A classic Snake Game built using **HTML, CSS, and JavaScript**. Control the snake, collect food, grow longer, and try to achieve the highest score without colliding with the walls or yourself.

## 🎮 Demo

> Clone the repository and open `index.html` in your browser to play.

---

## ✨ Features

- 🎯 Classic Snake gameplay
- 📐 25 × 25 game grid
- 🍎 Random food generation
- 📈 Real-time score tracking
- 🏆 High score saved using browser storage
- 💥 Game over detection
  - Wall collision
  - Self collision
- ⌨️ Keyboard controls
  - Arrow Keys
  - WASD Keys

---

## 🛠️ Tech Stack

- **HTML5** – Game structure
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Game logic and interactions

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/X-ImLucky-X/Snake_Game.git
```

### 2. Navigate to the project

```bash
cd snake-game
```

### 3. Run the game

Simply open **index.html** in your preferred web browser.

No additional setup or dependencies are required.

## 📖 How to Play

1. Start the game.
2. Control the snake using **Arrow Keys** or **WASD**.
3. Eat the food to grow longer and increase your score.
4. Avoid hitting:
   - The walls
   - Your own body
5. Try to beat your highest score!
